What is Focus Faux Pas?

Focus Faux Pas is a handy bookmarklet designed to help developers ensure that the focus appearance of various elements on a web page complies with accessibility guidelines. It checks elements like buttons, links, and more, making sure they're as sharp as a tack according to the Web Content Accessibility Guidelines (WCAG).

How Does it Work?

Focus Faux Pas calculates the minimum and actual focus areas for focusable elements, taking into account style details like outline width and offset. It then provides a visual indicator to show if each element's focus game is up to snuff or needs a little extra love.
